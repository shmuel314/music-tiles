רשימות השמעה

    שירים

    ייבוא מרובה: בחרו מספר קבצי שמע בבת אחת. שם השיר יילקח משם הקובץ, ללא תמונה.

      ייבוא תמונות מיוטיוב

      לחיפוש לפי שם שיר נדרש מפתח YouTube Data API (חינמי). התמונה נשמרת במכשיר בלבד — ללא ניגון מיוטיוב.

      יצירת מפתח: Google Cloud Console → YouTube Data API v3 → Credentials.

      אבטחה

      גיבוי ושיתוף

      צרו קובץ גיבוי אחד שניתן לשלוח להורה השני (וואטסאפ, מייל, Drive). השירים נשמרים רק במכשיר הזה — קובץ הגיבוי הוא הדרך היחידה להעביר אותם למכשיר אחר.

      ייצוא מלא כולל את קבצי השמע והתמונות (קובץ גדול, מומלץ לשיתוף). ייצוא מטא־דאטה כולל רשימות, שמות, סדר ותמונות בלבד — ללא קבצי שמע (קובץ קטן).

      הוספת שיר

      תמונה (לאריח)
      בחירת תמונה

      מחפש ביוטיוב לפי שם השיר, מוריד תמונה למכשיר. עובד ללא אינטרנט לאחר השמירה.
